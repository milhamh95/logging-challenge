## 11. Menambahkan Data Source

1. Masuk ke dashboard grafana -> localhost:3000
2. Login pakai credential username: admin, password: admin
3. Pilih menu connections > data sources
4. Klik tombol `Add new data source`
5. Isi name, untuk connection masukkan `http://prometheus:9090` (docker connection)
